| 情報の名前 | 内容・説明 | 例 |
| :--- | :--- | :--- |

| 商品の金額 | 一個あたりの値段 | １００円 |
| 個数 | 商品の個数 | １個 |
| 割引 | 商品一個に対する割引 ５０円引き |
| 税率の種類 |１０％、８％、免税のどれか | 税率は１０％です。|
